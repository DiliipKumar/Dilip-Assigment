# Terraform EC2 Module Setup

## Steps

1. **Terraform Commands**
   - Initialize:  
     ```bash
     terraform init
     ```
   - Plan:  
     ```bash
     terraform plan
     ```
   - Apply (with auto-approval):  
     ```bash
     terraform apply -auto-approve
     ```

2. **Destroy**
   - Destroy resources:  
     ```bash
     terraform destroy -auto-approve
     ```
